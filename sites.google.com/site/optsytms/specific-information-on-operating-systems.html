Informations spécifiques sur les systèmes d'exploitation

Principes de base du système d'exploitation
Un système d'exploitation est un programme informatique qui permet au matériel informatique de communiquer avec le logiciel. Un système d'exploitation est la partie la plus importante d'un appareil électronique.


Types of Operating Systems
Comme la technologie et les ordinateurs ont évolué, les systèmes d'exploitation aussi. Vous trouverez ci-dessous une liste des différents types de systèmes d'exploitation et une brève description de chacun. Avec la description est un exemple de chaque type de système d'exploitation. De nombreux logiciels ont plusieurs systèmes d'exploitation.  

GUI (Graphical User Interface) - Un système d'exploitation qui est généralement dirigé par une souris qui contient des graphiques et des icônes. Les systèmes d'exploitation sont beaucoup plus faciles à apprendre et à utiliser pour les utilisateurs finaux car les commandes n'ont pas besoin d'être connues ou mémorisées. Les systèmes d'exploitation GUI sont aujourd'hui les systèmes d'exploitation dominant pour les utilisateurs finaux. Exemple: Windows 98

Multi-utilisateur - Un système d'exploitation qui permet à un ordinateur d'être utilisé par plusieurs utilisateurs à la fois ou à des moments différents. Exemple: Linux

Multiprocessing - Un système d'exploitation capable de prendre en charge et d'utiliser plusieurs processeurs informatiques. Exemple: Windows 2000

Multitâche - Un système d'exploitation qui peut avoir plusieurs processus logiciels à exécuter simultanément. Exemple: Unix

Multithreading -  Systèmes d'exploitation qui permettent à différentes parties d'un logiciel de s'exécuter simultanément. Exemple: Unix, Windows 2000, Linux

Temps réel - Répond instantanément à l'entrée. 

Système d'exploitation un regard plus profond
Un système d'exploitation fournit une plate-forme logicielle dans laquelle les programmes d'application peuvent s'exécuter. Les programmes d'application doivent être écrits pour s'exécuter sur un système d'exploitation particulier. Votre choix de système d'exploitation détermine donc dans une large mesure les applications que vous pouvez exécuter. Un utilisateur interagit avec un système d'exploitation via un ensemble de commandes. Le système d'exploitation accepte ensuite les commandes via le processeur de commandes qui fait partie du système d'exploitation. Les utilisateurs entrent ces commandes via l'interface utilisateur graphique qui permet ensuite à l'utilisateur de cliquer sur les objets qui apparaissent à l'écran afin d'exécuter la commande. 

Système d'exploitation mobile
Comme la technologie a évolué, le système d'exploitation a évolué, les téléphones intelligents que beaucoup de gens utilisent aujourd'hui ont ce que vous appelleriez un système d'exploitation mobile. Ces systèmes d'exploitation sont spécialement conçus pour fonctionner sur les smartphones, PDA, tablettes et autres appareils portables. Le système d'exploitation est chargé de déterminer les fonctions et fonctionnalités disponibles sur votre appareil, telles que la molette, les claviers, le WAP, la synchronisation avec les applications, la messagerie électronique, la messagerie texte et plus encore. Un système d'exploitation mobile détermine également les applications que vous pouvez utiliser. à exécuter sur votre appareil portable. Certains des systèmes d'exploitation mobiles les plus populaires sont l'OS Android de Google, l'iPhone OS d'Apple et le Blackberry OS de Research in Motion.
